# my_first_project
プログラミング学習

space_rescue.py（「ゲームで学ぶPython！Pyxelではじめるレトロゲームプログラミング」リブロワークス［著］　北尾　崇［著・監修］　技術評論社　参考）

PythonとPyxelを使用して制作した、2Dアクション・レスキューゲームです。

プログラミング学習の第1作目として、基礎の習得と独自の課題解決を組み合わせて制作しました。

📝 プロジェクトの概要
このゲームは、書籍のサンプルコードをベースに、**「最新AI（ChatGPT）との協調」と「自身のルーツである秋田県の現状」**を反映させることをテーマに開発しました。

🛠️ 開発で工夫したポイント

1. 学習と技術の応用
• リファレンス: 「ゲームで学ぶPython！Pyxelではじめるレトロゲームプログラミング」の「space_rescue.py」を参考に、宇宙船の移動やオブジェクトの衝突判定といった基本ロジックを実装。

2. AI (ChatGPT) を活用した機能拡張
AIを単なるツールではなく「開発パートナー」と位置づけ、以下の機能を共同で実装しました。

• 動的な演出: 衝突時の爆発エフェクトのコード生成と調整。
• サウンドデザイン: ジェット音や救出時の効果音の最適な挿入タイミングの制御。
• システム構築: スコア・ライフ機能のロジック、およびプレイ中のUIレイアウトの設計。
• コード最適化: 実装したコードのバグ取りや、より効率的な書き方への修正。

3. 秋田県の要素と背景 
単なる装飾ではなく、秋田県の実際の情景や課題をゲームデザインに組み込みました。

• 設定の背景: 背景の宇宙空間や、宇宙船、宇宙飛行士は秋田県能代市にJAXAのロケット実験場があるということから、書籍のサンプルをそのまま採用しています。
• パワーアップアイテム: 秋田の食文化を象徴する**「きりたんぽ」**。取得することで一定時間無敵状態となります。
• 障害物としての「熊」: 秋田県内で深刻な社会問題となっている野生動物（熊）との遭遇を、宇宙空間の障害物として表現しました。単なるキャラクターではなく、地域が直面している脅威をゲーム内の緊張感として取り入れています。
